# ElasticSearch 基础

### 基本概念

`ElasticSearch` 简称 `ES`

`ES` 的特点：
分布式、全文检索、实时性

`ES` 和 `MYSQL` 类比：

| Mysql      | ES |
| ----------- | ----------- |
| Database      | Index       |
| Table   | Type        |
| Row   | Document        |
| Column   | Field        |
| Schema   | Mapping        |
